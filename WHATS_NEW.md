# What's New in OpenGalaxy About Dialog

## New Features Added

### System Information Display
Shows your system details:
- Operating System (e.g., Ubuntu 24.04.3 LTS)
- CPU Architecture (e.g., x86_64, arm64)
- Kernel Version (e.g., 6.8.0-51-generic)
- Qt Version (e.g., Qt 6.8.1)

### Check for Updates Button
- Click to open GitHub releases page
- See if new versions are available
- One-click access to downloads

### Enhanced Information
- Emphasizes multiplatform support
- Lists Windows, macOS, and Linux
- Correct GitHub repository
- Correct license (Apache 2.0)

## How to Access
1. Launch OpenGalaxy
2. Click "Settings" in sidebar
3. Scroll to "About" section
4. Click "About OpenGalaxy" button

## What You'll See
- OpenGalaxy icon (128x128)
- Version 1.0.0
- Full description
- Project links
- System information (NEW!)
- Check for Updates button (NEW!)

## Benefits
- Easy bug reporting with system info
- Quick update checking
- Better transparency
- More professional appearance

## Status
All features working and ready to use!
